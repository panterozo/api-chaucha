# Chaucha Api

Proyecto para subir la información de glosario [aquí](https://hackmd.io/HimeY1pvQwqpQMzb-2CfJQ) a una base de datos en MongoDB


## Instalación

```js
$ npm install
```

## Ejecución

```js
$ npm start
```

El proyecto hace su conexión mediante Mongoose hacia MongoDB.
